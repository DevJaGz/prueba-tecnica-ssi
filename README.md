## Installation


```sh
git clone  <url-repo>
cd frontend
npm i
ng serve
```

Then move to backend
```sh
cd ../backend
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 ./manage.py runserver
```
